The canonical Firestore rules now live in firestore.rules (repo root).

This file previously held six pasted iterations of the rules with no record
of which one was deployed. That history is preserved in git if you need it:
  git log -- "firebase rules.txt"

To deploy the canonical rules: copy firestore.rules into the Firebase console
(Firestore Database > Rules) or run `firebase deploy --only firestore:rules`.

NOTE: boxes created before 2026-07-11 have no creatorDeviceId field. The
rules in firestore.rules include a legacy fallback so they keep working;
new boxes created by the app now always include creatorDeviceId.
